## 2. Тестирование создания тендера

### 2.1 Создание тендера (шаг 1 - конcept)
- [ ] Подключить кошелек с ролью Member
- [ ] Перейти в раздел "Vytvoriť tender"
- [ ] Заполнить форму:
  - [ ] Názov tendru: "Test Tender"
  - [ ] Popis tendru: "Test description"
  - [ ] Maximálny rozpočet (EUR): 1000
  - [ ] Kategória: "Kancelárske potreby"
  - [ ] Dní otvorený pre ponuky: 7
  - [ ] Dní na hlasovanie: 3
- [ ] Нажать "Vytvoriť koncept tendru"
- [ ] Проверить, что появилось сообщение об успехе с ID тендера
- [ ] Проверить, что форма перешла на шаг 2

### 2.2 Публикация тендера (шаг 2)
- [ ] На шаге 2 проверить, что отображается информация о тендере
- [ ] Нажать "Uverejniť tender"
- [ ] Проверить, что появилось сообщение об успехе
- [ ] Проверить, что форма сбросилась
- [ ] Перейти в "Prehľad tendrov"
- [ ] Проверить, что тендер появился в списке со статусом "Aktívny"

### 2.3 Проверка валидации
- [ ] Попытаться создать тендер без названия - должна быть ошибка
- [ ] Попытаться создать тендер без бюджета - должна быть ошибка
- [ ] Попытаться создать тендер без категории - должна быть ошибка
- [ ] Попытаться указать дни < 3 или > 30 - должна быть ошибка
